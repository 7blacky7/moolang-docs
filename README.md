# moolang-docs

Dokumentation für [moolang](https://github.com/7blacky7/moo) (Alpha).

🌐 Live: <https://7blacky7.github.io/moolang-docs/>

## Lokal bauen

```sh
pip install mkdocs-material
mkdocs serve
# Öffne http://127.0.0.1:8000
```

Deployment erfolgt automatisch bei jedem Push auf `master` via GitHub Actions.
